2. **Fill in Metadata**

   - **Price**: Set $3.99 or $4.99 (see pricing section below)

   - **Image/Thumbnail**: [Optional] Create simple PNG with title + icon

3. **Paste Full Prompt Text**
   - Paste entire content from the prompt file (everything from "# API to Telegram Bot..." down to the Quality Checklist)
   - ✅ Include: System Message, User Message, Few-Shot Example, Quality Checklist

**Q: Should I use code blocks in the prompt text?**
A: Yes! Code blocks render well on PromptBase and show concrete output. Keep them.

**Q: Can I sell the same prompt on multiple platforms?**
A: Yes, but disclose it in description. PromptBase doesn't require exclusivity.

**Q: How long until first sale?**
A: 3-7 days typically (PromptBase takes time to index). Don't panic if nothing sells first week.

**Q: What if a prompt gets bad reviews?**
