✨ KEY FEATURES
═══════════════════════════════════════════════════════════════════════════

安全防護 (Safety Pack)
  ✓ 防止危險 Bash 指令 (rm -rf /, git push --force 等)
  ✓ 保護敏感檔案 (.env, credentials, token 等)
  ✓ 禁止自動 git push (防止未審查代碼推送)

品質控制 (Quality Pack)
  ✓ 強制先讀後改 (大檔案 >100 行警告)
  ✓ 自動代碼風格檢查 (Python ruff/pylint, JavaScript eslint)

效率提升 (Productivity Pack)
  ✓ 改動 >5 檔案時提醒 commit

通用特性
  ✓ 一鍵安裝 (互動或命令行)
  ✓ 3 種安裝模式 (全裝 / 安全 / 品質 / 效率)
  ✓ 自動備份原設定
  ✓ 即時生效 (無需重啟)
  ✓ 易於禁用或自訂

🚀 INSTALLATION METHODS
═══════════════════════════════════════════════════════════════════════════

方法 1: 互動式 (推薦新手)

  # 選擇要裝的套裝，按 Enter

方法 2: 全裝
  $ python3 install.py --all

  $ python3 install.py --safety      # 只裝安全防護

🔧 TECHNICAL SPECS
═══════════════════════════════════════════════════════════════════════════

系統需求
  • Python 3.6+ (for install.py)
  • Bash 4.0+ (for hook scripts)
  • macOS / Linux / WSL
  • 家目錄可寫入

Hook 事件類型
  • PreToolUse   (執行前檢查，可阻止)
  • PostToolUse  (執行後檢查，可反饋)

Exit Code 約定
  • 0: 通過
  • 1: 警告 (提示但不阻止)
  • 2: 阻止 (操作被取消)

效能指標
  • 平均執行時間: 50-200ms (除 linter 外)
  • 最大 timeout: 5000ms (可自訂)
